# Design System: Premium GTM Intelligence

## 1. Visual Theme & Atmosphere
The design system follows a "Data Intelligence" aesthetic. It utilizes deep blacks, muted grays, and neon cyan accents to create a sense of professional precision. High contrast for data points, soft shadows for container depth.

## 2. Color Palette & Roles
- **Deep Space Background:** (#0a0b10) Main page surface.
- **Cyber Surface:** (#14161e) Card and dashboard containers.
- **Data Neon Blue:** (#00f2ff) Primary accents, active HTMX triggers, and link highlights.
- **Positive Pulse:** (#00ff88) Up-trends and "True" GTM classifications.
- **Neutral Silver:** (#e0e0e0) Primary body text.
- **Ghost Gray:** (#707070) Secondary metadata and labels.

## 3. Typography Rules
- **Font Family:** 'Inter', sans-serif. Use 'JetBrains Mono' for data values and code snippets.
- **Headers:** Semi-bold, tight letter spacing (-0.02em).
- **Body:** Regular weight, generous line height (1.6) for readability.

## 4. Component Stylings
- **Cards:** Subtly rounded (12px), background: Cyber Surface, thin border (#1e222d).
- **Buttons:** Sharp edges with 2px corner radius, background: Data Neon Blue (solid) for CTAs.
- **Tables:** Minimalist, no vertical borders, alternate row shading.

## 5. Layout Principles
- **Whitespace:** Generous guttering (24px) between dashboard widgets.
- **Grid:** 12-column flexbox based layout.
